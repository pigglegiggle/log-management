## โคลนโปรเจกต์
```bash
git clone https://github.com/pigglegiggle/log-management
```

## เข้าไปในโฟลเดอร์โปรเจกต์
```bash
cd log-management
```

## ตั้งค่าทุกอย่างอัตโนมัติ
```bash
./Makefile/run.sh
```

## เข้าใช้งานหน้าเว็บ

`http://localhost:3001/`


## ลองส่ง log เข้าไปใน Project

```javascript
cd samples && node post_log.js
```
